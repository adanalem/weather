# weather
🌦️ Dynamic Weather Pro A sleek, modern weather dashboard built with HTML5, CSS3 (Tailwind), and Vanilla JavaScript. This app provides real-time weather updates with a focus on immersive user experience through dynamic animated backgrounds.
✨ Features
Real-time Data: Fetches live weather information (Temperature, Humidity, Wind Speed) using the OpenWeatherMap API.

Dynamic Backgrounds: Features a custom-built Animated Gradient Engine that changes the entire theme of the UI based on the current weather condition (Clear, Rain, Clouds, Snow).

Glassmorphism UI: A premium "Frosted Glass" card design using backdrop-filter and Tailwind CSS.

Fully Responsive: Optimized for both desktop and mobile viewing.

Interactive UX: Smooth CSS transitions and keyframe animations for a high-end feel.

🛠️ Tech Stack
Frontend: HTML5, Tailwind CSS

Logic: JavaScript (ES6+)

API: OpenWeatherMap API
